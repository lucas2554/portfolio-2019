Goal :

For this project we had to create a complete graphic identity for a luxurious chalet near Gerardmer, France. This graphic identity includes a logo, a benchmarking, and a website wireframe for the chalet. The main goal of the projects was to creare a full graphic chart usable for the website of the chalet.

What I have learned :

With this project, I learned how to use Adobe Indesign in a professional way. We had to be really attentive to the consistency between the graphic chart we create and the the graphic identity of the chalet.
